## 🚀 Getting Started

### Prerequisites

- Modern web browser
- Basic understanding of time formats

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/VinayShetyeOfficial/JS-Watch.git
   ```
2. Navigate to the project directory:
   ```sh
   cd JS-Watch
   ```
3. Open in browser:
   ```sh
   open index.html
   ```
